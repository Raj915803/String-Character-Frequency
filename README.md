# String-Character-Frequency
Java coding
# Approach
1. Take the input string.
2. Use a LinkedHashMap to store characters and their counts.
3. LinkedHashMap maintains insertion order, so characters appear in the order of first occurrence.
4. Traverse the string character by character.
5. For each character:
6. If it already exists in the map → increment count.
7. Otherwise → add it with count = 1.
8. Finally iterate through the map and print results.
